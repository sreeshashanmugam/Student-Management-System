# Student Management System (C++)

## Description
A console-based application to manage student records using C++ and file handling.

## Features
- Add student
- Display students
- Update student
- Delete student
- Persistent storage using file handling

## Technologies Used
- C++
- fstream

## How to Run
1. Open in Code::Blocks
2. Build & Run (F9)
